# 4장 요약: 정보 시스템과 전략

## 1. 정보 시스템의 개념

- 정보 시스템(IS)은 조직의 의사결정과 업무 수행을 지원하는 시스템  
- 하드웨어, 소프트웨어, 데이터, 사람, 프로세스로 구성됨  
- 목적: 효율성 향상, 경쟁력 확보, 전략적 의사결정 지원

## 2. 정보 시스템의 유형

- **거래처리시스템(TPS)**: 반복적이고 일상적인 거래 처리 (ex. 급여, 주문)
- **경영정보시스템(MIS)**: TPS 데이터를 요약해 관리자에게 보고
- **의사결정지원시스템(DSS)**: 비정형 문제 해결을 위한 분석 도구 제공
- **전략정보시스템(SIS)**: 경쟁우위 확보를 위한 전략적 시스템
- **지식관리시스템(KMS)**: 조직 내 지식 공유 및 활용 촉진

## 3. 정보 시스템과 조직 전략

- 정보 시스템은 조직의 전략과 밀접하게 연결됨
- **Porter의 경쟁 전략**:
  - 비용우위 전략
  - 차별화 전략
  - 집중화 전략
- 정보 시스템은 각 전략을 지원하는 도구로 활용됨

## 4. 가치사슬(Value Chain)과 정보 시스템

- **가치사슬**: 제품/서비스가 고객에게 전달되기까지의 활동 흐름
- **주요 활동**: 생산, 마케팅, 판매, 서비스 등
- 정보 시스템은 각 활동의 효율성을 높이고 연결성을 강화함

## 5. 정보 시스템의 전략적 역할

- 경쟁우위 확보: 빠른 정보 처리, 고객 맞춤 서비스 제공
- 혁신 촉진: 새로운 비즈니스 모델 창출
- 조직 구조 변화: 업무 방식, 의사결정 구조 변화 유도

## 6. 사례 분석

- **Amazon**: 물류 시스템 자동화로 비용 절감 및 고객 만족도 향상
- **Netflix**: 추천 알고리즘으로 사용자 경험 개선
- **Zara**: 실시간 재고 관리로 빠른 트렌드 대응

## 7. 정보 시스템 도입 시 고려사항

- 조직의 전략과 일치 여부
- 기술적 적합성
- 사용자 수용도
- 비용 대비 효과
